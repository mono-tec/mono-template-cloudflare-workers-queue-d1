# 1. リスク評価基準

## 影響度

| 評価 | 内容 |
|------|------|
| 高 | プロジェクト継続に大きな影響を与える |
| 中 | 一部機能やスケジュールへ影響する |
| 低 | 軽微な影響で対応可能 |

---

## 発生確率

| 評価 | 内容 |
|------|------|
| 高 | 発生する可能性が高い |
| 中 | 発生する可能性がある |
| 低 | 発生する可能性は低い |

# 2. 特記事項

本プロジェクトは、

- 個人開発
- AI支援開発
- 技術検証

を目的としているため、

特に以下リスクを重点管理対象とする。

- R-001 スコープクリープ
- R-005 開発担当者の体調不良
- R-006 AI支援環境への依存
- R-010 ドキュメント未整備による知識喪失

上記リスクについては、
フェーズ完了時に見直しを実施する。
